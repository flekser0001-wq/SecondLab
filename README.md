# Task Manager
Простое приложение для управления задачами на Python (Tkinter + JSON).

## Запуск
1. Установите Python 3.x  
2. Запустите файл `gui.py`
3. Добавляйте и удаляйте задачи через интерфейс

## Функциональность
- Добавление/удаление задач  
- Сохранение задач в JSON-файл  
- Автоматическая загрузка при старте
