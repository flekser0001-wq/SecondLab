# Task Manager
Небольшое приложение для управления списком задач на Python с использованием Tkinter и JSON.

## Запуск
1. Установите Python версии 3.x  
2. Откройте и запустите файл `gui.py`  
3. Работайте с задачами через удобный графический интерфейс

## Возможности
- Добавление и удаление задач  
- Сохранение списка в JSON-файл  
- Автоматическая загрузка задач при запуске программы
